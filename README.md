Charu's Kitchen – QR Code Menu Website

Overview

Charu's Kitchen QR Menu Website ek simple aur mobile-friendly digital menu hai.
Customers banner ya poster par lage QR code ko scan karke direct restaurant ka menu dekh sakte hain.

Website HTML5, CSS aur JavaScript ka use karke banayi gayi hai aur kisi bhi browser me easily open ho sakti hai.

---

Features

- Mobile friendly design
- Search food item feature
- Food images display
- WhatsApp order button
- Zomato availability mention
- Clean restaurant style UI
- QR code compatible menu page

---

Technologies Used

- HTML5 – Page structure ke liye
- CSS3 – Styling aur responsive layout ke liye
- JavaScript – Menu search functionality ke liye

How to Run

1. "index.html" file download karein.
2. File ko kisi bhi browser (Chrome, Edge, Firefox) me open karein.
3. Menu website screen par open ho jayegi.

---

How to Create QR Menu

1. Website ko hosting platform par upload karein (Netlify / GitHub Pages).
2. Website ka URL copy karein.
3. Kisi QR generator se us URL ka QR code generate karein.
4. QR code ko banner, poster ya table card par print karwa dein.

Customers QR scan karte hi menu website open ho jayegi.
